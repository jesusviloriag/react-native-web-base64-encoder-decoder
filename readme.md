# React Native Base64 Encoder & Decoder

Install dependencies
```
npm install
```

Run in Android
```
npx react-native run-android
```

Run in iOS
```
npx react-native run-ios
```

Run in Web
```
npm run web
```

Compile Web in Windows
```
npm build-windows
```

Compile Web in Unix systems (Linux & Mac)
```
npm build-unix
```
